这个百度地图api呢是学长介绍给我的，用户通过ak等参数来向百度地图api发送GET请求，百度地图api会根据请求参数的不同来返回不同的json结果，可以说是很方便了，只需要调用参数和截取

这个东西是这么用的（没用过的童鞋看过来(。・∀・)ノ）

要先创建应用来获取ak（每一个用户的每个应用a值k都不一样）记住这里的ak值，我们以后获取数据都得用这个ak

接下来最好进行开发者认证，这个认证对个人开发者不要钱（只要你请求的数据量不是太大），上个图感受一下认证的好处( •̀ ω •́ )y

然后我们就可以开始请求数据啦，先上个网址

Web服务API | 百度地图API SDK​lbsyun.baidu.com/index.php?title=webapi/guide/webservice-placeapi

这里用地点检索来做说明
